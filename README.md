Student Management Platform

This repository contains a **Student Management Platform** built to manage student information efficiently. It allows you to add, update, view, and delete student records. The platform can handle details such as student names, IDs, courses, grades, and contact information. It’s a simple yet practical project for learning and practicing CRUD operations, database management, and interface design.

**Features:**

* Add new student records
* Update existing student information
* Delete student records
* View all students in a structured format
* Search and filter students by ID or name
